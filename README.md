# Java Notes by Sayan

Этот репозиторий содержит мои личные заметки по Java, Spring и другим инструментам, таким как Redis, Kafka, и Minio. Эти заметки включают основные концепции, примеры кода и лучшие практики для удобного и эффективного использования этих технологий. Я использовал репозиторий Maslovsky-dev/JavaNotes в качестве основы и добавил свои заметки и примеры.

## Основы Java

Содержит основные концепции Java, такие как работа с переменными, типами данных, операторами, условными операторами и циклами. Примеры кода и объяснение базовых принципов для начинающих и продолжающих.

## Java Collections

Описание коллекций в Java (List, Set, Map), а также реализация и примеры использования различных коллекций. Углубление в особенности коллекций и их типичные сценарии применения.

## Concurrency в Java

Основы многопоточности в Java, включая использование потоков, задач и более сложные концепции, такие как ExecutorService, CompletableFuture, синхронизация и java.util.concurrent.

## Java 17

Особенности Java 17, на котором построен проект. Обзор новых возможностей версии и примеры использования последних обновлений языка.

## Spring Core

Основные концепции Spring Framework, включая внедрение зависимостей (DI), инверсию управления (IoC) и контекст приложения. Также содержатся примеры конфигурации и внедрения компонентов.

## Spring Boot

Использование Spring Boot для создания и развертывания приложений с минимальной конфигурацией. Примеры настройки проекта, работы с профилями и использования встроенного сервера Tomcat.

## Spring Data

Заметки по Spring Data, включая работу с репозиториями, реализацию CRUD-операций и использование JPQL/HQL-запросов. Информация о работе с PostgreSQL и других базах данных.

## Spring Security

Основы Spring Security, настройка аутентификации и авторизации в приложении, реализация JWT для защиты REST API. Примеры базовой и продвинутой конфигурации безопасности.

## Работа с Redis

Введение в Redis как систему кэширования и базы данных ключ-значение. Примеры конфигурации Redis в Spring Boot, интеграции и использования Redis Template для кэширования данных.

## Взаимодействие с Kafka

Заметки по интеграции Apache Kafka для обмена сообщениями между микросервисами. Настройка и конфигурация продюсеров и потребителей Kafka, примеры применения для передачи событий в реальном времени.

## Использование Minio для загрузки и удаления файлов

Работа с Minio для хранения и управления файлами. Примеры загрузки файлов в Minio, удаление и взаимодействие с Minio API для управления файлами в приложении.

## Тестирование

Подходы к тестированию в проекте, включая модульное тестирование, интеграционное тестирование с использованием MockMvc, тестирование с базами данных (H2) и использование Testcontainers для контейнеризированного тестирования.

## Лицензия

Этот проект создан для личного использования и обучения. Вы можете свободно использовать и изменять материалы по своему усмотрению.
